# UI Function Map

## Shell

- One HTML document.
- Four hidden/visible sections controlled by JS.
- Bottom navigation switches `data-view`.
- URL does not change.
- Browser back/forward does not represent section changes.

## Mobile Screens In Scope

- Home.
- Works as a screen structure only, not content quality.
- Profile as a screen structure only.
- Bottom navigation.

## Out Of Scope Screens / Areas

- Desktop.
- Booking screen content.
- Gallery content quality.

## Core UI Controls

- Hero CTA: switches to booking screen.
- Final CTA: switches to booking screen.
- Bottom nav buttons: switch sections.
- Telegram links: external links.
