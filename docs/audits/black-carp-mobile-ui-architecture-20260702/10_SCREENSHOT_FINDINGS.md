# Screenshot Findings

## Image 1

- Three unrelated contexts are visible at once: work caption, approach paragraph, price block.
- Approach text is too long for display serif and breaks badly.
- Price heading is clipped by viewport/nav, so the page looks broken.
- Dividers and texture make the transition heavy instead of editorial.

## Image 2

- Hero to editorial transition has a large dead zone.
- `GRAPHIC STUDIES` feels like a template label, not a necessary part of the composition.
- Editorial heading has poor line breaks, especially the orphaned `и`.
- Huge serif text is used again, so hierarchy becomes monotonous.

## Image 3

- Info strip looks like an admin checklist, not a premium editorial block.
- Number column, dividers, and bold sans text are too utilitarian.
- Final CTA repeats the same oversized serif + horizontal-line pattern.
- Bottom nav visually competes with the CTA.

## Root Causes

1. Display serif is used for too many jobs.
2. Long Russian text is not manually composed.
3. Sections do not own clean viewport moments.
4. Global grain sits over text and UI.
5. UI dividers and nav are too heavy for the intended quiet luxury tone.
